Realms 教程


Minecraft 和 Microsoft 账户
在 Xbox、Windows 或 Android 和 iOS 移动设备上畅玩 Minecraft 时，您可以使用 Microsoft 账户登录，使用 Xbox 玩家代号保存您的进度、购买内容以及使用 Realms 与您的朋友在线畅玩。

 

注意：您仍然需要购买 XBOX LIVE 金会员订阅才能在 Xbox 主机上在线畅玩。在其他设备（例如 PlayStation 4 和 Nintendo Switch）上，您可以使用免费的 Microsoft 账户登录 Minecraft，但是仍然需要相应的订阅服务（例如 PLAYSTATION PLUS 或 NINTENDO SWITCH ONLINE）才能在线畅玩。

 

通过免费的 Microsoft 账户，您可以访问所有 Minecraft 设备上的购买内容、Realms 和 其他 Minecraft 内容。Microsoft 账户可以免费设置，并在任何 Minecraft 设备上使用。

 

 

设置 Microsoft 账户
如需设置免费的 Microsoft 账户，请遵循以下说明：

在 Minecraft 中，选择登录。
输入您想使用的账户电子邮件地址。
输入您想使用的账户密码。  注意：密码必须至少为 8 个字符，并且至少包含大写字母、小写字母、数字和符号中的两种组合。
使用下拉菜单选择国家/地区，然后设置您的出生日期。
查看您的电子邮箱获取 Microsoft 发送的验证码。输入或粘贴验证码，然后选择下一步。
 

您的 Microsoft 账户现已设置完毕！您将在 Minecraft 中自动登录您的账户。

 

注意：如果您未满 13 岁，则需要获得家长许可才能完成账户设置。请让家长帮助您激活 Microsoft 账户。

 

通过免费的 Microsoft 账户，您可以访问所有 Minecraft 设备上的购买内容、Realms 和 其他 Minecraft 内容。Microsoft 账户可以免费设置，并在任何 Minecraft 设备上使用。

 

 

在 PlayStation 上使用 Microsoft 账户登录
如需登录您的免费 Microsoft 账户，请遵循以下说明：

在 Minecraft 中，选择使用 MICROSOFT 账户登录。
在其他设备上，打开网页浏览器并导航至 Minecraft 中显示的 URL。输入 Minecraft 中显示的代码，然后单击下一步。
输入 Microsoft 账户的电子邮件地址、电话号码或 Skype 用户名，然后单击下一步。
输入密码，然后单击登录。
登录之后，Minecraft 会显示欢迎消息。选择开始游戏！。
 

 

在 Nintendo Switch 上使用 Microsoft 账户登录
如需登录您的免费 Microsoft 账户，请遵循以下说明：

在 Minecraft 中，选择使用 MICROSOFT 账户登录。
在其他设备上，打开网页浏览器并导航至 Minecraft 中显示的 URL。输入 Minecraft 中显示的代码，然后单击下一步。
输入 Microsoft 账户的电子邮件地址、电话号码或 Skype 用户名，然后单击下一步。
输入密码，然后单击登录。
登录之后，Minecraft 会显示欢迎消息。选择开始游戏！。
 

 

Windows
如果您在 Windows 商店中使用玩家代号购买游戏，则 Minecraft 将自动使用此玩家代号登录。如需使用其他账户登录，请遵循以下说明：

启动 Xbox 应用程序。
单击档案和设置菜单，然后单击注销。
单击档案和设置菜单，然后单击登录。
在弹出的窗口中，单击登录，然后通过您想使用的玩家代号登录。
启动 Minecraft。如果您使用玩家代号登录失败，请确保您在 Xbox 应用程序中使用正确的玩家代号登录，然后重启 Minecraft。
 

 

Xbox
Minecraft 将自动使用您当前 Xbox 档案的玩家代号登录。

 

 

成人、青少年和儿童账户的区别
在 Minecraft 中，家长可以为儿童和青少年账户设置以下活动许可：

加入多人游戏
接收邀请加入其他玩家的游戏
通过语音或文本沟通
购买包或 Realms/Realms Plus 订阅
如需通过 Realms 在线畅玩，您需要配置设备隐私设置以允许 Minecraft 通过网络进行通信，并确保俱乐部和多人游戏设置都设为已允许。
