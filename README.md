想设置多人游戏服务器？

请注意：该服务器的设置仅与 Minecraft:Java Edition 兼容。

如果您想自己运行 Minecraft 多人游戏服务器，该流程可能会有些复杂（请参阅该 Wiki 文章了解相关教程）。首先，请确保您可以通过命令行使用 Java。该项在 Linux 和 macOS 系统中应该已经设置好了，但在 Windows 系统中您可能需要对 PATH 环境变量进行调整。

下载 minecraft_server.1.21.10，并使用以下命令运行：

                                            java -Xmx1024M -Xms1024M -jar minecraft_server.1.21.10.jar nogui
										如果您想使用图形用户界面启动服务器，您可以省略“nogui”部分。

如你所知，在此页面下载任何软件，即表示你同意 Minecraft 最终用户许可协议和隐私政策。

如果您想跳过设置，开始探索其他第三方服务器，请访问我们的官方服务器列表网站。每个服务器都有其乐趣和独特之处。在我们的服务器列表网站中寻找您最喜欢的服务器，所有列出的服务器均已通过审核并已确认其符合我们的社区标准及指南。根据游戏类型、游玩风格等浏览服务器。无论您多大年纪，游玩方式如何，总有一款服务器适合您。
